Variables Naming

Use snake_case for variables, methods and symbols

variable_name = :some_symbol

def some_method
  ...
end

Use CamelCase for classes and modules

class SomeClass
  ...
end

Use SCREAMING_SNAKE_CASE for other constants

SOME_CONST = 5
